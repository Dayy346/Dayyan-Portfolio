# QA screenshots

Screenshots used as evidence in the visual audit tickets. All files are generated by Playwright and kept in this repo.

## What’s here

- **01–05:** Boot, login, desktop, start menu, taskbar
- **06–16:** Individual app windows (About, Resume, Projects, Experience, Skills, Power, LeetCode, Contributions, Contact, Chatbot, Help)
- **17–20:** Desktop with one/many windows, taskbar stress, desktop icons closeup
- **21–23:** Mobile (390px login, 390px lite, 320px lite)
- **24–25:** Window chrome (title bar buttons, About content area)

## Regenerating

From the repo root:

```bash
npx playwright test tests/qa-screenshots.spec.ts
```

To see the browser while capturing:

```bash
npx playwright test tests/qa-screenshots.spec.ts --headed
```

Screenshots are written to this folder. Ticket references are in:

- `docs/TICKETS-QA-AUDIT-ROUND2.md` (Round 2 — thorough audit with picture refs)
- `docs/TICKETS-QA-AUDIT.md` (Round 1)
